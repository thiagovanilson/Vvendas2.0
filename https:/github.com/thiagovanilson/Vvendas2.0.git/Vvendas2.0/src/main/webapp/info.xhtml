V-Vendas 1.0



Software desenvolvido para fins academicos apenas.
Desenvolvido por Thiago Vanilson Florencio.
Cursando Superior em Analise e Desenvolvimento de Sistemas,
na cidade de Monteiro - PB.

As seguintes tecnologias foram aplicadas no desenvolvimento deste produto: JAAS, JPA, PrimeFaces, JSF.

Este software tem o proposito de gerenciar os produtos de alguns tipos de comercios,
realizando vendas e facilitando o gerenciamento de informações, automatizando assim algumas realidades.

Requisitos do sistema

* Java 7 ou superior.
* My SQL server.
* Navegador web (Recomendado Chrome acima da versão 45).
* Wildfly ou outro servidor web.
